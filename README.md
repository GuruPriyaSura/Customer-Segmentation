# 🧀 Customer Segmentation & Repayment Risk
**Author:** Gurupriya R | Mphasis
**Impact:** Flagged default-risk accounts before missed payments

## Overview
K-Means (k=4) segments 1,200 customers into Premium, Stable, At-Risk, High-Risk. Gradient Boosting flags 60%+ default probability accounts.

## Run
```bash
pip install -r requirements.txt
python customer_segmentation.py
```
